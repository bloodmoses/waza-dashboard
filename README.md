# WAZA Track Club Results Dashboard

Static HTML dashboard for track and field results.

## How to Update

1. Update the `data to add.xlsx` file with new results
2. Run: `python generate_dashboard.py`
3. Commit and push:
   ```bash
   git add index.html
   git commit -m "Update results"
   git push
   ```

The dashboard will automatically update on waza.pearseprojects.org

## Local Development

Open `index.html` in your browser to preview changes.
